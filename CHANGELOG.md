# v0.1.0

- Countdown from a given duration, or count up indefinitely
- Flexible time input: `90`, `90s`, `5m`, `1h`, `1h30m`, `1h30m20s`
- Colour changes give a quick visual read of remaining time:
  - **Countdown** - green &rarr; yellow &rarr; red as time drains
  - **Count-up** - cycles through the same spectrum
  - **Paused** - dim amber
  - **Finished** - flashing green
- Colon blinks once per second while running
- Resizable window
- Controls: `SPACE` (start/pause), `R` (reset), `ESC` (quit)
